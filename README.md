# Foodie 🍽️

Foodie is a Spring Boot based backend system for a modern food delivery application.  
It provides REST APIs for managing users, restaurants, menus, and orders.

---

## 🚀 Features

- User Management
- Restaurant Management
- Menu Handling
- Order Processing
- RESTful APIs
- DTO Architecture
- Layered Architecture
- MySQL Database Integration
- Spring Data JPA
- Lombok Support

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- hibernate
- RestAPI
- Lombok

---

## ▶️ Run Locally

Clone the project

```bash
git clone https://github.com/your-username/
food-delivery-backend.git
```

Move into project directory

```bash
cd food-delivery-backend
```

Run the application

```bash
mvn spring-boot:run
```
